# 📘 Soft Computing Study App

A modern, interactive **web-based study platform** designed to help students master **Soft Computing concepts** including Neural Networks, Fuzzy Logic, and Genetic Algorithms — all in one place.

---

## 🚀 Features

- 📚 **Structured Study Modules**
  - Module I: Neural Networks & Fuzzy Basics  
  - Module II: Advanced Fuzzy Logic & Genetic Algorithms  

- 📝 **Detailed Study Notes**
  - Concept explanations
  - Examples and formulas
  - Easy-to-understand breakdowns

- 🧮 **Formula Cheat Sheet**
  - 40+ important formulas
  - Quick revision before exams

- ❓ **Q&A Practice Bank**
  - 30+ exam-oriented questions
  - Categorized by topics
  - Includes answers for revision

- 🎯 **Exam Preparation Section**
  - Important topics
  - Time management strategy
  - Common mistakes

- 📊 **Progress Tracker**
  - Checklist-based learning
  - Visual progress bar

- 🔍 **Search & Filter Functionality**
  - Quickly find topics or questions

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript  
- **UI Design:** Custom responsive design  
- **Fonts:** Google Fonts (Syne, DM Sans, JetBrains Mono)
